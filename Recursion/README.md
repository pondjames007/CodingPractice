* Reduce: from functools import reduce -> [#726](./726_NumOfAtoms.py)
* minmax: 可選的一方要在選項中選擇將其優勢最大化的選擇，另一方則選擇令對手優勢最小化的方法
    -> 如果可以贏的話一定會選能贏的那個 -> [#464](./464_CanIWin.py)
* Recursion時 需要track 整個string/array 放一個pos進去recursion function
